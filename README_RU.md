# CleanLine 2.0 Demo - сайт + PWA-приложение

Это готовый демонстрационный пакет, который можно показать владельцу CleanLine как альтернативный современный сайт и приложение.

## Что внутри

- `index.html` - современная витрина магазина.
- `styles.css` - адаптивный дизайн под телефон и компьютер.
- `app.js` - демо-каталог, корзина, заявки, PWA-установка, AI-чат-демо.
- `manifest.webmanifest` - файл приложения для Android/PWA.
- `sw.js` - service worker для установки и офлайн-режима.
- `pages/` - SEO-страницы: стирка, Gallus Moldova, опт, B2B.
- `docs/` - аудит, коммерческое предложение, SEO-план, Android-план.
- `tools/make_qr.py` - скрипт генерации QR-кода после деплоя.

## Как открыть локально

1. Распакуйте архив.
2. Откройте папку в терминале.
3. Запустите:

```bash
python3 -m http.server 8080
```

4. Откройте в браузере:

```text
http://localhost:8080
```

PWA-установка работает на `localhost` или на HTTPS-домене после деплоя.

## Как показать клиенту

1. Открыть главную страницу.
2. Показать блок “Боли текущего сайта”.
3. Показать каталог, корзину и заявку.
4. Показать B2B-страницу.
5. Показать PWA-установку на телефон.
6. Открыть SEO-страницы в папке `pages/`.
7. Предложить пакет: сайт + PWA + SEO + CRM + AI.

## Что нужно для реального запуска

- Доступ к домену или новый поддомен.
- Экспорт товаров из текущего сайта или Excel.
- Фото товаров.
- Условия доставки и оплаты.
- Доступы к Google Analytics 4 и Google Search Console.
- Для Android в Play Market: аккаунт Google Play Developer.
