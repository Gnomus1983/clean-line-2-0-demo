# SEO-план CleanLine на 90 дней

## Месяц 1 - фундамент

- Анализ текущих страниц.
- Новые title/description для главной, категорий и товаров.
- Настройка sitemap.xml и robots.txt.
- Подключение Google Analytics 4 и Google Search Console.
- Страницы: средства для стирки, средства для уборки, средства для посуды, бытовая химия оптом.

## Месяц 2 - брендовые страницы

- Gallus Moldova.
- ONYX Moldova.
- Power Wash Moldova.
- Passion Gold Moldova.
- Balea Moldova.
- Alio Moldova.
- Внутренняя перелинковка между брендами и категориями.

## Месяц 3 - контент и B2B

- Статьи: как выбрать гель для стирки, чем убрать жир, как выбрать средство для ванной.
- B2B-страницы для кафе, офисов, клининга, автомоек.
- FAQ-разметка.
- Анализ запросов из Search Console и расширение семантики.
