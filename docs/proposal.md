# Коммерческое предложение для CleanLine

## Цель проекта

Создать современный сайт и PWA-приложение для CleanLine, которые будут не просто показывать товары, а увеличивать заявки, повторные заказы и оптовые продажи.

## Что будет сделано

### 1. Новый сайт-витрина

- современная главная страница;
- удобный каталог;
- корзина;
- быстрый заказ в WhatsApp/Telegram;
- акции, хиты, готовые наборы;
- страницы доставки, оплаты, возврата;
- блок доверия и отзывы.

### 2. PWA-приложение

- установка на экран Android;
- иконка приложения;
- быстрый доступ к каталогу;
- офлайн-кэш базовых страниц;
- будущие push-уведомления;
- QR-код для установки.

### 3. SEO

- структура SEO-страниц;
- брендовые страницы: Gallus Moldova, ONYX Moldova, Power Wash Moldova;
- категория: средства для стирки, уборки, посуды;
- страница: бытовая химия оптом;
- sitemap.xml, robots.txt;
- микроразметка Schema.org;
- подключение Google Search Console.

### 4. CRM и заявки

- заявки с сайта в CRM;
- заявки в Telegram менеджеру;
- история клиентов;
- повторные продажи;
- сегментация розница/опт.

### 5. AI-консультант

- помогает подобрать товар;
- отвечает по доставке и оплате;
- собирает заявку;
- передаёт менеджеру.

## Пакеты

### Старт - от 700 евро

Главная страница, базовая PWA-установка, базовое SEO.

### Бизнес - от 1500 евро

Новый сайт, каталог, корзина, PWA, SEO-страницы, аналитика.

### Премиум - от 3000 евро

Сайт, PWA, CRM, AI-консультант, автоматизация, аналитика продаж.

## Главная выгода

CleanLine получает не новый дизайн, а систему, которая помогает привлекать клиентов из Google, получать больше заявок и возвращать покупателей через приложение и CRM.
