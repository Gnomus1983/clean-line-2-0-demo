# Аудит Clean-line.md и стратегия предложения

## Главный вывод

Текущий сайт - рабочий каталог бытовой химии, но он выглядит как шаблонная витрина. Его можно усилить по четырём направлениям: SEO, конверсия, B2B-продажи и мобильное приложение.

## Что уже есть

- Каталог товаров.
- Категории: средства для стирки, посуда, уборка, уход, гигиена, хозтовары.
- Цены и корзина.
- Контакты, WhatsApp/Telegram, форма обратного звонка.
- Русская и румынская версии.

## Слабые места

1. Слабый первый экран: нет сильного оффера, цифр, доверия и B2B-призыва.
2. Категории слишком короткие: "Средства для стирки" вместо "Средства для стирки из Европы в Кишинёве - купить с доставкой по Молдове".
3. Нет отдельных SEO-страниц под бренды: Gallus Moldova, Power Wash Moldova, ONYX Moldova, Balea Moldova.
4. Нет сильной страницы "Бытовая химия оптом".
5. Мало доверия: отзывы, фото склада/шоурума, реальные кейсы, цифры.
6. Нет готовых наборов, которые повышают средний чек.
7. Нет видимой CRM-логики: клиент пришёл, ушёл - его сложно вернуть.
8. Нет сильной PWA/Android-логики: иконка на телефоне, push, быстрый повторный заказ.

## Что предложить

Позиционировать услугу так: не "сайт", а "система роста продаж интернет-магазина бытовой химии".

Состав решения:

- Новый современный сайт.
- PWA-приложение для Android.
- SEO-страницы под категории, бренды и опт.
- CRM для заявок и повторных продаж.
- AI-консультант для подбора товара.
- Аналитика GA4, Search Console, Meta Pixel.

## Самый сильный заход

"Я посмотрел ваш сайт. У вас хороший ассортимент, но сайт теряет бесплатный трафик, оптовых клиентов и повторные продажи. Я подготовил демо, как может выглядеть новая система: сайт, приложение, SEO, CRM и AI-консультант".
